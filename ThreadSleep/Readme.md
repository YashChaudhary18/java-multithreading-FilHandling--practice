Java Thread Sleep Example

Overview :

This program demonstrates the use of "Thread.sleep()" in Java.

The thread prints numbers from 1 to 5 and pauses for 1 second between each number.

---

Steps :

1. Create a class extending the "Thread" class.
2. Override the "run()" method.
3. Print numbers from 1 to 5.
4. Use "Thread.sleep(1000)" to pause the thread for 1 second.
5. Handle "InterruptedException" using try-catch.
6. Start the thread in the main method.

---

Concepts Used :

- Java Multithreading
- Thread class
- "Thread.sleep()"
- Exception handling
